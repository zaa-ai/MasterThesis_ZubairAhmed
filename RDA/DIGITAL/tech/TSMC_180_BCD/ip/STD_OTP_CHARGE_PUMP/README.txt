1) The milkyway in /mw/ is generated by Elmos with the tool Milkyway.
   First change name of metal layer in lef-File to M1/M2/M3/M4.
   Start the tool: 
   module load milkyway
   milkyway -tcl &
   Copy to the tool command line:
   create_mw_lib -tech ./src/sc9tap_cm018mg_base_rvt.tf \
              -bus_naming_style {[%d]} \
              ips_tsmc180bcd50_p1r_ad_LIB

   read_lef -lib_name ips_tsmc180bcd50_p1r_ad_LIB \
         -cell_lef_files ./src/ips_tsmc180bcd50_p1r_ad_modified.lef \
         -cell_version overwrite
         
   read_lef -lib_name ips_tsmc180bcd50_p1r_ad_LIB \
         -cell_lef_files ./src/ips_tsmc180bcd50_p1r_ad_ant_modified.lef \
         -cell_version overwrite
